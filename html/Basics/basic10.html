Basic Text Formatting

This webpage showsaces the use of various text formatting options
available in html

  1. Bold: Use < b >...< /b > tag to make text bold
  2. Italic: Use < i>...< /i> tag to make text italic
  3. Underline: Use < u>...< /u> tag to underline text
  4. Strike: Use < strike>...< /strike> tag to create a strike over text
  5. Monospace: Use < tt>...< /tt> tag to make text use monospace font
  6. Superscript: Use < sup>...< /sup> tag to superscript a string
  7. Subscript: Use < sub>...< /sub> tag to subscript a string
  8. Insert: Use < ins>...< /ins> tag to insert some text. Inserted text will be underlined
  9. Delete: Use < del>...< /del> tag to delete piece of text
  10. Enlarge Text: Use < big>...< /big> tag to enlarge a piece of text
  11. Reduce Text: Use < small>...< /small> tag to delete piece of text
